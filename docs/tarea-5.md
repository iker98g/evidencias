# 5. Servidor FTP

Lo primero que vamos a hacer es habilitar el puerto 21 en el FTP.

![](images/tarea-5/puerto_ftp.PNG)

A continuación, crearemos los tres siguientes usuarios: cliente, servidor y administrador.
